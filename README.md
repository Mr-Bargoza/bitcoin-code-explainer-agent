# Bitcoin Code Explainer

This project is a simple Python-based AI agent that helps beginners understand
Bitcoin-related code by explaining common keywords such as transactions, blocks,
verification, and signatures.

## Why this project?
Bitcoin Core code can be difficult for new contributors to understand.
This tool aims to reduce the learning curve by providing simple explanations
in plain English.

## Tech Used
- Python
- Regular Expressions (regex)

## Target Users
- Beginner Bitcoin developers
- Students interested in open-source Bitcoin development

This project is built as a learning tool and does not modify or interact with
the Bitcoin network.
## How to Run

Run the program using the command below:

```bash
python bitcoin_code_explainer.py
